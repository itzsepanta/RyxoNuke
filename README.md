## 📖 Overview

**Ryxo Nuke** is a Python CLI tool for **Discord server administration** — channel & role management, member tools, server info, and moderation helpers — all through a beautiful Rich terminal interface.

## 🚀 Quick Start

### 1. Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/ryxo-nuke.git
cd ryxo-nuke

# Run installation scripts (Windows)
python_installer.bat     # Checks Python installation
setup.bat                # Installs dependencies
start.bat                # Launches the CLI
```

### 2. Bot Setup

1. Go to [Discord Developer Portal](https://discord.com/developers/applications)
2. Create a new application → Bot
3. **Enable Required Intents:**
   - ✅ Server Members Intent
   - ✅ Message Content Intent
4. Copy your bot token
5. Invite bot to your server with needed permissions

## 🎮 Navigation

|    Key    | Action            |
| :-------: | ----------------- |
|    `n`    | Next page         |
|    `b`    | Previous page     |
| `01`–`39` | Execute a command |
|    `q`    | Quit application  |
